# Digital Ledger

A premium, glassmorphic personal finance manager designed to help you track credits, debits, and lending history with ease. Built with modern web technologies, it features a secure PIN system, dynamic charts, and a beautiful visual interface.

## Features

- **Secure Authentication**: PIN-based login and registration system to keep your data private.
- **Glassmorphism UI**: A stunning, modern translucent design with dynamic background gradients.
- **Transaction Tracking**:
  - **Credit**: Record money coming in (e.g., salary, gifts).
  - **Debit**: Track daily expenses with categories (Food, Bills, Shopping, etc.).
  - **Lend/Borrow**: Manage debts and loans with friends, including "Settle" options.
- **Visual Analytics**: Interactive charts to visualize spending habits.
- **Dark Mode**: Fully supported premium dark theme.
- **Responsive Design**: Optimized for both mobile and desktop experiences.
- **Local Storage**: Data is persisted securely in your browser's local storage.

## Tech Stack

- **Frontend**: HTML5, CSS3 (Modern Variables & Flexbox/Grid)
- **Scripting**: Vanilla JavaScript (ES6+ Modules)
- **Libraries**:
  - [GSAP](https://greensock.com/gsap/) - For smooth animations and transitions.
  - [Chart.js](https://www.chartjs.org/) - For data visualization.
  - [Remix Icon](https://remixicon.com/) - For high-quality icons.

## Getting Started

### Prerequisites

You need a modern web browser to run this application. No backend server is required as it runs entirely client-side.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/digital-ledger.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd digital-ledger
    ```
3.  Open `index.html` in your browser.
    *   Alternatively, for the best experience, use a local server:
        ```bash
        npx serve .
        ```

## Usage

1.  **Register**: On first launch, enter your name and create a 4-digit PIN.
2.  **Dashboard**: View your total balance, "In Hand" cash, and "Bank" balance.
3.  **Add Transaction**: Use the **Credit**, **Debit**, or **Loan** buttons to add records.
4.  **View History**: Click "History" to see a detailed list of all transactions.
5.  **Settings**: Click your profile icon to toggle Dark Mode.
